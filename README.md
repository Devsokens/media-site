# The Daily Chronicle

Your trusted source for breaking news, in-depth analysis, and insightful journalism covering world events, technology, culture, and business.

## Getting Started

### Prerequisites

- Node.js & npm installed

### Installation

1. Clone the repository
2. Install dependencies:
   ```sh
   npm install
   ```
3. Start the development server:
   ```sh
   npm run dev
   ```

## Technologies Used

- Vite
- TypeScript
- React
- shadcn-ui
- Tailwind CSS
